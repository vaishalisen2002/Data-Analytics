# Data-Analytics
This repository contains a curated set of quantitative data analytics and reporting projects aligned with real-world business and financial use cases.

The projects demonstrate hands-on experience in collecting, transforming, and analyzing large datasets using SQL and Python, building dashboards in Power BI, and translating analytical outputs into actionable insights for stakeholders.

Use cases include financial risk and variance analysis, customer churn prediction, and time-series forecasting to support strategic and operational decision-making.
